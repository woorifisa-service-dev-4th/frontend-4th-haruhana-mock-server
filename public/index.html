🌱 하루하나 API 문서

문제 관련 API

GET /api/problems

전체 문제 목록을 조회합니다.

Query Parameters:

  • category: 특정 카테고리의 문제만 조회
  • level: 특정 난이도의 문제만 조회

GET /api/problems/categories

전체 카테고리 목록을 조회합니다.

통계 관련 API

GET /api/statistics/summary

사용자의 전체 통계 요약 정보를 조회합니다.

GET /api/statistics/weekly

주간 문제 해결 현황을 조회합니다.

GET /api/statistics/category

카테고리별 통계 정보를 조회합니다.
